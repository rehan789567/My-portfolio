## 📌 About This Project

This is my personal developer portfolio website — designed to showcase my skills, projects, and background as a 17-year-old aspiring developer from Multan, Pakistan.

> **⚡ Transparency Statement:**
> I built this website with the help of **Claude AI** (by Anthropic) to test and check its efficiency in generating real-world, production-quality frontend code. This was an experiment to see how well AI can assist a beginner developer in building a fully functional, modern portfolio — and the results were impressive.

---

## ✨ Features

- 🌑 **Dark Neon Theme** — Cyan/blue glassmorphism aesthetic
- 🎯 **Custom Cursor** — Glowing dot with smooth trailing ring
- 🌌 **Animated Particle Background** — Interactive connected particle canvas in the hero
- ⌨️ **Typewriter Effect** — Rotating role titles in the hero section
- 📜 **Scroll Reveal Animations** — Smooth fade-in as you scroll
- 📊 **Animated Skill Bars** — Progress bars that animate on scroll into view
- 🔢 **Counter Animations** — Numbers that count up when the about section is visible
- 📱 **Fully Responsive** — Works on mobile, tablet, and desktop
- 📋 **Copy Email Button** — One-click email copy in the contact section
- 🚀 **Zero External Dependencies** — Single HTML file, works offline instantly

## 🚀 How to Run

**Option 1 — Local (Instant):**
```
1. Download index.html
2. Double-click it
3. Opens in your browser — done!
```

**Option 2 — GitHub Pages (Free hosting):**
```
1. Upload index.html to a GitHub repo as index.html
2. Go to Settings → Pages → Branch: main → Save
3. Live at: https://rehan789567.github.io/portfolio
```

**Option 3 — Netlify Drop:**
```
1. Go to drop.netlify.com
2. Drag and drop index.html
3. Get an instant live link
```
